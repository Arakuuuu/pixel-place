# Pixel/Place

A collaborative pixel grid marketplace. 1,000,000 pixels. Buy, paint, trade.

## Features
- 1000×1000 pixel grid (1 million pixels)
- Canvas-based rendering with zoom & pan
- Buy unclaimed pixels for coins
- Paint your pixels any color
- List pixels for sale on the marketplace
- Trade pixels with other users (10% fee on resales)
- Real-time updates
- Leaderboard
- GitHub + Email authentication

## Stack
- Next.js
- Supabase (database, auth, realtime)
- Vercel (hosting)
